# 损失厌恶

对于损失产生的厌恶 **远超过** 对收益产生的期望。

[亚洲疾病问题](https://zh.wikipedia.org/wiki/%E6%A1%86%E6%9E%B6%E6%95%88%E6%87%89)

[框架效应 - 维基百科，自由的百科全书](https://zh.wikipedia.org/wiki/%E6%A1%86%E6%9E%B6%E6%95%88%E6%87%89)

设定参考点：

人在面对 **收益** 时是 **风险厌恶** 的， ->  股票赚到钱就想出手(把风险收益转为确定收益)。

人在面对 **损失** 时是 **风险偏好** 的， ->  （喜欢冒险以避免损失出现）股票亏钱了还期待反弹，不想平仓。 （[[sunk_cost|沉没成本]]）

## 框架效应

1.风险选择框架 (风险，确定)

面对收益，面对损失

2.特性框架(积极，消极)

积极描述使人开心，消极描述使人不开心。

eg 手术风险： 90% 成功（开心）
手术风险： 10% 死亡（难过）

eg 果汁： 90% 果肉（开心）
果汁： 10% 化学添加剂（难过）


3.目的性框架(收益，损失)

迎合人们对收益的期望。

eg 加油站：5元/升 信用卡手续费额外加0.6元    加油站：5.6元/升 现金支付立省0.6元

加收手续费是一种 **损失** ，人们不喜欢损失。 而立省是一种 **收益** ，迎合人们对收益的期望。

买车险送小礼品，想当于价格加收益，迎合人们对收益的期望，（而不是直接打折）。

买糕点称重(2kg)，一个是先称多点(2.2kg)再一点点取出（敏感！取出的部分是损失！）一个是先称少点(1.8kg)再一点点添上（愉快，加上的部分是收益）
