# Windows

### 备份及恢复本机的IP信息

  * 备份本机的IP信息： `netsh -c interface dump > network_config.txt`
  * 恢复本机的IP信息： `netsh -f network_config.txt` 或者（`netsh -c Interface -f network_config.txt`）
