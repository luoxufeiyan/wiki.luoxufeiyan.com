## 实用工具

* 系统性能监控
```
brew cask install stats
```

* 壁纸自动换
```
brew cask install paper
```

* 压缩
```
brew install --cask keka
```

* 键盘分享
```
brew install --cask synergy
```

* rescuetime
```
brew install --cask rescuetime
```

* LyricsX 歌词
```
brew install --cask lyricsx
```
