# IDLE配置优化

IDLE是Python的自带编辑器，在某些环境下可能是唯一可用的IDE。

## 配置高亮

IDLE自带的电几个主题高亮种类都不全，作罢

## 快捷键

增加多行缩进： Ctrl + ]  (右中括号)

减少多行缩进：Ctrl + [  (左中括号)

添加多行注释：Alt + 3

取消多行注释：Alt + 4

变量联想： Alt + /

函数补全： Ctrl + Shift + Space
